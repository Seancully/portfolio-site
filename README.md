# Sean Culleton Portfolio

Personal portfolio site built from scratch with plain HTML, CSS, and JavaScript.

## Live Site

https://seancully.github.io/portfolio-site/

## Run Locally

From this folder, start a static file server:

```bash
python3 -m http.server 5500
```

Then open:

http://localhost:5500/

## Project Structure

- index.html: main page markup
- styles.css: site styles and responsive layout
- main.js: interaction logic and effects
- 404.html: GitHub Pages not-found page
- assets/: images, project thumbnails, and media

## Deployment

The site is published with GitHub Pages from the main branch.
